# Практика от компании Zuzex

## Веб-сервис – социальная сеть ИТ-специалистов

### Требования:

#### Авторизация и регистрация

Создать страницу, где пользователь может либо войти в существующий аккаунт, либо создать новый. При регистрации пользователь заполняет имя, фамилию, уникальный никнейм, email, пароль с подтверждением и выбирает роль из списка: Frontend Developer, Backend Developer, QA Engineer, Designer, Manager, HR. При входе пользователь указывает никнейм и пароль.

#### Лента постов

Главная страница приложения — лента публикаций. Посты отображаются карточками с заголовком, именем автора, превью текста, типом поста, фото-превью и счётчиком лайков. Фильтрация по типу поста и направлению. Создание, редактирование и удаление постов доступно авторизованным пользователям, редактирование и удаление — только автору. Лайки может ставить любой авторизованный пользователь.

#### Профиль пользователя

Страница профиля доступна всем авторизованным пользователям. Отображает имя, фамилию, никнейм, роль, место работы и описание, а также портфолио с проектами.

#### Стек технологий

**Фронтенд:** Vite, React, TypeScript, Redux, SCSS.

**Бэкенд:** Express, MongoDB, Mongoose, bcrypt, JWT.
